# Twitter-Sentiment-Analysis

## Description of Dataset
* Dataset used : Twitter US Airline Sentiment Dataset
* The dataset consists of 14640 tweets divided into 3 categories as positive, neutral or negative stored in a csv file.

## Approach
* Imported the csv file
* Cleaned the tweets
* Created a train and test data in standard format
* Used Multinomial Naive Bayes for classification

## Results
* Accuracy of 77.59%
